# Intermediate Level Projects 🚀

Collection of intermediate Python projects focusing on data structures, file handling, and application development.

## Projects

### 📋 Contact Book (`contact_book.py`)
A command-line contact management application.

**Concepts:**
- Dictionaries for data storage
- User-defined functions
- Menu-driven interfaces
- Data manipulation (CRUD operations)

**Features:**
- Add new contacts
- View all contacts
- Edit existing contacts
- Delete contacts
- Interactive menu system

**Usage:**
```bash
python contact_book.py
```

---

### 📊 Gradebook (`gradebook.py`)
Application for managing and tracking student grades.

**Concepts:**
- Data structures (lists, dictionaries)
- Grade calculations and statistics
- File I/O operations
- Data validation

**Features:**
- Add student grades
- Calculate averages
- Track performance metrics
- Generate reports

---

### 📄 JSON Practice (`json_practice.py`)
Working with JSON data format for data persistence.

**Concepts:**
- JSON serialization and deserialization
- File I/O with JSON
- Data format conversion
- External data handling

**Topics:**
- Reading JSON files
- Writing JSON data
- Parsing JSON structures
- Data transformation

---

## Running Projects

### Prerequisites
- Python 3.7+

### Execute a Project
```bash
cd intermediate
python <filename>.py
```

### Example
```bash
python contact_book.py
python Student_Management_System_v1.py
```
