# Intermediate Level Projects

This folder contains practical Python applications that build on the beginner exercises.

## Projects

### Contact Book
File: [contact_book.py](contact_book.py)

A simple CRUD-style command-line contact manager that uses dictionaries and menus.

### Gradebook
File: [gradebook.py](gradebook.py)

A small tool for storing student grades and calculating averages.

### JSON Practice
File: [json_practice.py](json_practice.py)

A short example showing how to read and write JSON data.

### Student Management System
Files: [Student_Management_System_v1.py](Student_Management_System_v1.py) and [utility.py](utility.py)

A more complete management tool for student records and grade tracking.

### School Management System
File: [school_management_system.py](school_management_system.py)

A broader example for managing school-related records and operations.

### DNS Lookup Tool
File: [Dns_lookup.py](Dns_lookup.py)

A simple command-line script that resolves a domain and checks common subdomains using the Python socket module.

## Running examples

```bash
python projects/intermediate/contact_book.py
python projects/intermediate/Dns_lookup.py
```

## Learning focus

- Dictionaries and structured data
- File handling and JSON
- User input and validation
- Building small apps from scratch
