# Python Projects

Collection of Python projects demonstrating core programming concepts and practical applications.

## Projects Overview

### 🧮 Simple Calculator (`simple_calculator.py`)
An interactive calculator application with continuous operation support.

**Features:**
- Addition, Subtraction, Multiplication, Division
- Input validation for operations and numbers
- Division by zero error handling
- Continuous operation until user exits
- User-friendly menu interface

**Usage:**
```bash
python simple_calculator.py
```

**Key Concepts:**
- Function definitions
- Error handling (try/except)
- User input validation
- Loop control (while/break)

---

### 👨‍🎓 Student Management System (`Student_Management_System_v1.py` & `utility.py`)
A system for managing student records and grades.

**Features:**
- Add new students with grades
- Calculate average grades
- Find highest grade
- Store student data
- Menu-driven interface

**Usage:**
```bash
python Student_Management_System_v1.py
```

**Key Concepts:**
- Data structures (lists, dictionaries)
- Global variables
- Function organization
- Statistical calculations (sum, max, average)
