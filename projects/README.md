# Project Portfolio

This folder contains Python projects grouped by skill level so the learning path is easy to follow.

## Beginner level

Focus areas: syntax, functions, loops, input/output, and small exercises.

Examples:
- [beginner/simple_calculator.py](beginner/simple_calculator.py)
- [beginner/functions_practice.py](beginner/functions_practice.py)
- [beginner/challenge.py](beginner/challenge.py)

## Intermediate level

Focus areas: data structures, file handling, JSON, CRUD-style applications, and command-line tools.

Examples:
- [intermediate/contact_book.py](intermediate/contact_book.py)
- [intermediate/gradebook.py](intermediate/gradebook.py)
- [intermediate/school_management_system.py](intermediate/school_management_system.py)
- [intermediate/Student_Management_System_v1.py](intermediate/Student_Management_System_v1.py)
- [intermediate/Dns_lookup.py](intermediate/Dns_lookup.py)

## Advanced level

Focus areas: object-oriented programming and system-style design.

Example:
- [advanced/Library_management_system.py](advanced/Library_management_system.py)

## Running projects

```bash
python projects/beginner/simple_calculator.py
python projects/intermediate/contact_book.py
python projects/advanced/Library_management_system.py
```

## Suggested learning path

1. Start with beginner exercises.
2. Move to intermediate applications.
3. Practice advanced object-oriented design.
4. Keep improving the code and adding features.
