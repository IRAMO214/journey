# YSec Toolkit v1.2 Roadmap

This file tracks the planned improvements for the next release of YSec Toolkit.

## Planned features

- [ ] Colored output using colorama
- [ ] Progress indicator while scanning
- [ ] Better JSON reports
- [ ] Configuration file (config.json)
- [ ] Logging support
- [ ] Automatic report naming with timestamps

## Goal

Make the toolkit more polished, easier to use, and better for long-running scans.
